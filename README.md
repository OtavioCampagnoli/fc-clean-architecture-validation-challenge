# Clean Architecture - Api layer challenge

## Sobre este Projeto

Este repositorio contém um desafio de implementação api layer seguindo os princípios de Clean Architecture. O objetivo é criar uma estrutura de código que seja fácil de manter, escalável e que proteja o domínio da aplicação.

## Pré-requisitos

- Node.js versão 16 ou superior

## Instalação

```bash
npm i
```

## Executando os Testes

```bash
npm test
```

## Desafio de Implementação

- Criar uma API RESTful para gerenciar um recurso de produtos.

- **Testes Unitários**: Validar a lógica de negócio isolada
- **Testes de Integração**: Validar a comunicação entre camadas

### Objetivos

- Proteger o domínio da aplicação
- Manter baixo acoplamento entre camadas
- Orientar o código para casos de uso explícitos
- Garantir facilidade em desenvolvimento, deploy, operação e manutenção
